# Final-Project
```
Setup
0. Install git and CMake (sudo apt install)
1. sudo apt update
2. sudo apt install \
    libxrandr-dev \
    libxcursor-dev \
    libxi-dev \
    libudev-dev \
    libfreetype-dev \
    libflac-dev \
    libvorbis-dev \
    libgl1-mesa-dev \
    libegl1-mesa-dev \
    libfreetype-dev
3. cd Final-Project
4. cmake -B build
5. cmake --build build
6. Finally, you can use ./build/bin/main to run the code
```
